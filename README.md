## 🎮 快速开始

```bash
# 克隆本仓库（虽然你可能只是看看）
git clone https://github.com/Dawn-Jovi/PersonalWeb.github.io.git

# 安装依赖（其实大部分时间在解决依赖冲突）
npm install

# 运行项目（祈祷不要报错）
npm start
```

> 💡 温馨提示：如果遇到问题，可以尝试以下步骤：
>
> 1. 删除 node_modules
> 2. 重新 npm install
> 3. 重复以上步骤直到放弃
