
## 🎮 快速开始

```bash
# 克隆本仓库（虽然你可能只是看看）
git clone https://github.com/Dawn-Jovi/my-site.git

# 安装依赖（其实大部分时间在解决依赖冲突）
npm install

# 运行项目（祈祷不要报错）
npm start
```
